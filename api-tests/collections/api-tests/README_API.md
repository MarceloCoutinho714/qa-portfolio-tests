#  API Tests - Portfolio QA

##  Objetivo
Este projeto contém testes de API realizados utilizando Postman para validar endpoints de autenticação, busca e fluxo principal da aplicação.

##  Ferramentas utilizadas
- Postman
- JavaScript (scripts de teste)
- JSON

##  Estrutura

api-tests/
├── collections/
├── evidence/
└── README_API.md

##  Como executar os testes

1. Abrir o Postman
2. Importar a collection presente em:
api-tests/collections/
3. Executar a collection completa

##  Cenários testados

- Login com credenciais válidas
- Login com credenciais inválidas
- Validação de status code
- Validação de estrutura da resposta
- Fluxo principal da API

##  Resultado

Todos os testes foram executados com sucesso, validando o comportamento esperado da API.
